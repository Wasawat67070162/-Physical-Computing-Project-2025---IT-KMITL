## ALAN
Alan หรือ Anti lost alarm notice เป็นอุปกรที่จะเเจ้งเตือนคุณ
เมื่อระยะของสิ่งของที่คุณติดเอาไว้ห่างกับมือถือของคุณตามระยะที่กำหนด
เพื่อช่วยเหลือผู้คนที่ชอบลืมของสำคัญของตัวเอง 
มีข้อดีตรงที่ ขนาดเล็ก, พกพาใช้งานง่าย, รองรับมือถือทุกระบบ ทำเองได้ถูกกว่า ของที่ขายเป็น Anti-Lost Tracker สำเร็จรูป ปรับแต่งได้หลายระดับ: ตั้งระยะเตือน, แจ้งเตือนผ่านมือถือ
เเละง่ายต่อการใช้งาน
## จุดประสงค์
เพื่อให้สามารถเเจ้งเตือนผู้ใช้เวลาของสำคัญหายไปได้
## แนวคิดการทำงานของระบบ

ระบบจะใช้ Bluetooth Low Energy (BLE) เป็นตัวกลางในการเชื่อมต่อระหว่างอุปกรณ์ และ โทรศัพท์มือถือ เมื่อโทรศัพท์อยู่ในระยะที่กำหนด ระบบจะถือว่า เชื่อมต่อปกติ

แต่เมื่อการเชื่อมต่อถูกตัดขาด เช่น
- โทรศัพท์ถูกวางทิ้งไว้ไกลเกินระยะ
- อุปกรณ์ถูกนำออกไปจากพื้นที่
- การยกเลิกการเชื่อมต่อโดยตั้งใจ

ระบบจะทำงานดังนี้:

โทรศัพท์มือถือจะแสดงการแจ้งเตือนทันที เพื่อเตือนผู้ใช้ว่าอุปกรณ์หลุดออกนอกระยะปลอดภัย
อุปกรณ์ฝั่งฮาร์ดแวร์จะทำการแจ้งเตือนด้วยเสียงจาก Buzzer พร้อมแสดงสถานะผ่าน LED / LCD (ขึ้นกับการออกแบบ)

กล่าวคือ ระบบนี้ทำงานทั้ง แจ้งเตือนบนมือถือ และแจ้งเตือนที่อุปกรณ์จริง เพื่อป้องกันการสูญหาย เช่น
- กระเป๋าเดินทาง
- อุปกรณ์สำคัญต่าง ๆ
